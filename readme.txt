Version 1.0

Da ich es auf immer mehr seiten sehe und man bei mir schon öfter mal den wunsch geäußert hat eine Scrollschrift auf der HP auszugeben ohne diese immer in der html zu bearbeiten.
--------------------------------------------------------------

FEATURE:

- Administrierung komplett im Adminbereich
- erstellte Texte können gespeichert werden oder auch nicht
- Texte können in jederform von große, farbe usw bearbeitet 	werden
- es können auch alle Texte deaktiviert werden so das es auch 	keine Ausgabe auf der Seite gibt.
--------------------------------------------------------------

HINWEIS:

man kann die Felder auch leerlasen dann nimmt es die Standartwerte, diese sind wie folgt aufgebaut:

Standartschriftfarbe: #FF0000 (red)
Standartschriftgröße: 1
Standartscrollgeschwindigkeit: 3
--------------------------------------------------------------

INSTALLATION:

- alle dateien in die Ordnerstruktur hochladen
	(es werden keine vorhanden dateien überschrieben)
- installation ausführen:
	www.deine-homepage.de/index?installation
- in der index.htm eures designs an gewünschter stelle die box 	einbauen. Dazu einfach diesen Texte 

{_boxes_gfa_scrolltext}

	im gewünschten bereich der index.htm platzieren :)
--------------------------------------------------------------

FEHLER:

Derzeit sind noch keine Fehler bekannt.
